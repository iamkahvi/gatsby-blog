# kahvi's blog
Rebuild of my [Jekyll blog](https://github.com/iamkahvi/iamkahvi.github.io) with Gatsby.

## Install and Usage
1.  **Install packages.**

    ```sh
    npm install
    ```
2.  **Start developing.**

    ```sh
    gatsby develop
    ```

## Built With
* [Gatsby](https://www.gatsbyjs.com/)
* [Tachyons](https://tachyons.io/)
* [Ghost CMS](https://ghost.org/)
